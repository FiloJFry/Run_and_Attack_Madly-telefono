# Run and Attack Madly
Run and Attack Madly è uno sparatutto in prima persona incentrato sulle battaglie boss. È un gioco molto essenziale: 0% robaccia da tossici,0% pay-to-win,100% skill,1000% tryhard. Divertiti a sconfiggere i boss più assurdi con le armi più assurde nell'ambiente più assurdo che tu abbia mai visto! 
- GIOCO NON ANCORA TERMINATO
- Ricorda di estrarre le immagini dalla cartella "Immagini.zip" prima di iniziare a giocare!
# Ok ho installato tutto,ora come faccio a giocare?
- Scarica SimpleHTTPserver sul telefono
- Dall'app,clicca su "Root folder"
- Clicca il menù a tendina in alto e cerca la cartella di gioco precedentemente scaricata (il menù a tendina apre le cartelle del telefono)
- Una volta inserita la cartella con "select this folder",torna indietro e clicca su "start"
- Cerca su Google,Safari o qualunque altro motore di ricerca http://localhost:8080 In questo modo,dovresti ritrovarti tutti i file di gioco disposti in colonna N.B.: per far ciò non è necessario il Wi-Fi
- Se non funziona,prova a cercare http://localhost:8000 o assicurati di aver premuto "start" prima
- Una volta trovata la schermata con tutti i file di gioco,fai doppio click su SchermataAvvio.html per aprire il gioco
- Buon divertimento!
